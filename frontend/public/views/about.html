About ft_transcendence

Project Overview

ft_transcendence is a comprehensive web-based platform featuring real-time Pong gameplay. This 42 school project demonstrates our ability to create a complete web application with modern technologies, featuring user authentication, tournament systems, and multiplayer capabilities.

Our implementation includes classic Pong, AI-powered gameplay with multiple difficulty levels, a four-player brick breaker mode, and a tournament system with customizable settings. The platform offers responsive design, user profiles, match history tracking, and a visually appealing interface.

Technology Stack

Frontend

  • CSS

Backend

  • JWT Authentication

Database

DevOps

Key Features

Multiple Game Modes

Classic Pong, AI opponent, four-player brick breaker, and tournament system

Intelligent AI

Adaptive computer opponent with three difficulty levels for solo play

User Management

Secure authentication, profiles, game history, and win/loss statistics

Game Customization

Adjust colors, winning scores, and game parameters for a personalized experience

Secure Authentication

Local account creation and Google OAuth integration

Multilingual Support

Available in English, French, and Spanish with easy language switching
