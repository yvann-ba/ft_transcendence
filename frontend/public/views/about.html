About ft_transcendence

Project Overview

ft_transcendence is a web-based real-time Pong game platform developed as part of the 42 school curriculum. This project challenges us to create a complete web application featuring user authentication, real-time gameplay, tournament systems, and social features.

Our implementation features both local and AI gameplay, customizable game settings, user profiles, and a sleek, modern interface designed to provide an exceptional gaming experience.

Technology Stack

Frontend

  • TypeScript
  • Tailwind CSS
  • Vite

Backend

  • Fastify (Node.js)
  • WebSockets
  • JWT Authentication

Database

  • SQLite
  • Custom ORM

DevOps

  • Docker
  • Nginx
  • SSL/TLS
