How We Built ft_transcendence

Planning Phase

We started by breaking down the requirements and planning our architecture. This involved choosing technologies, designing database schemas, and creating wireframes.

Development Phase

With a clear plan in place, we built the core features iteratively. Starting with the basic Pong game, we added authentication, user profiles, and multiplayer capabilities.

Polishing Phase

After implementing the core functionality, we enhanced the UI/UX, optimized performance, and implemented additional features like AI opponents and customization options.

Deployment Phase

Finally, we containerized the application with Docker, set up Nginx for routing, and implemented security measures for a production-ready deployment.
